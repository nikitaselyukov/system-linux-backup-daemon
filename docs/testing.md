# Тестирование

Рекомендуемые сценарии проверки:

1. `SOURCE` существует, `TARGET` существует — копирование выполняется успешно.
2. `TARGET` отсутствует — каталог создаётся автоматически.
3. `SOURCE` не существует — процесс завершается с ошибкой, запись видна в журнале.
4. В исходном каталоге есть вложенные папки — они копируются рекурсивно.
5. Ручной запуск через `snapshotctl run` создаёт новую копию.
6. Изменение `FREQUENCY` через `snapshotctl set frequency ...` изменяет расписание таймера.
7. После перезагрузки системы таймер остаётся активным.
